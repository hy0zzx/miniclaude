주요 기능Features

Claude Code를 더 편하게 Work more comfortably with Claude Code

🔔

즉각적인 알림Instant alerts

Claude Code 응답 완료 또는 권한 확인 요청이 오는 순간 캐릭터가 점프하며 말풍선으로 알려줘요. 다른 창을 보고 있어도 놓치지 않아요.

The moment Claude Code finishes or needs input, the character jumps with a speech bubble. Never miss a response even while in another window.

📊

사용량 대시보드Usage dashboard

캐릭터를 클릭하면 현재 세션 사용량과 이번 주 Claude 모델별 토큰 사용량을 한눈에 확인할 수 있어요. 5분마다 자동 갱신됩니다.

Click the character to view current session usage and this week's token usage by model. Auto-refreshes every 5 minutes.

🖱️

자유로운 배치Place anywhere

위젯을 드래그해서 모니터 어디에든 배치할 수 있어요. 자리를 옮기면 대시보드도 따라와요.

Drag the widget to any position on your monitor. The dashboard follows along when you move it.

⚡

자동 시작 & 훅 등록Auto-start & hooks

설치 시 Windows 시작 프로그램과 Claude Code 훅이 자동으로 등록돼요. 설정 없이 바로 쓸 수 있어요.

Startup registration and Claude Code hooks are auto-configured at install time. Zero manual setup needed.

🕐

시간대별 인사Time-based greetings

아침·점심·저녁·새벽 각각 다른 메시지로 인사해줘요. 오래 작업하는 날도 조금 더 즐겁게.

Different messages for morning, afternoon, evening, and late night. A small touch that makes long sessions more enjoyable.

💰

토큰 비용 제로Zero token cost

사용량 대시보드는 Claude OAuth 방식의 Usage API를 사용해요. Claude API를 직접 호출하지 않아 토큰이 전혀 소모되지 않아요.

The usage dashboard calls Anthropic's OAuth-based Usage API — not the Claude API itself — so it consumes zero tokens.
