🔔

즉각적인 알림

Claude Code 응답 완료 또는 권한 확인 요청이 오는 순간 캐릭터가 점프하며 말풍선으로 알려줘요. 다른 창을 보고 있어도 놓치지 않아요.

📊

사용량 대시보드

캐릭터를 클릭하면 현재 세션 사용량과 이번 주 Claude 모델별 토큰 사용량을 한눈에 확인할 수 있어요. 5분마다 자동 갱신됩니다.

🖱️

자유로운 배치

위젯을 드래그해서 모니터 어디에든 배치할 수 있어요. 자리를 옮기면 대시보드도 따라와요.

👻

잠깐 숨기기

캐릭터를 클릭하면 5초간 반투명으로 사라져요. 화면이 복잡할 때 잠깐 비켜달라고 할 수 있어요.

⚡

자동 시작 & 훅 등록

설치 시 Windows 시작 프로그램과 Claude Code 훅이 자동으로 등록돼요. 설정 없이 바로 쓸 수 있어요.

🕐

시간대별 인사

아침·점심·저녁·새벽 각각 다른 메시지로 인사해줘요. 오래 작업하는 날도 조금 더 즐겁게.

설치 방법

설치는 30초면 충분해요

1

인스톨러 실행

HopClaude-Setup-1.0.0-win.exe를 다운로드해서 실행하세요

2

자동 시작 체크

설치 중 "Windows 시작 시 자동 실행" 체크 유지를 권장해요

3

완료!

위젯이 바로 실행되고 Claude Code 훅도 자동 등록돼요
